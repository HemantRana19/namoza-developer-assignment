# Namoza Developer Assignment

This repository contains my submission for the Namoza Developer Assignment. The project demonstrates frontend development, Google Tag Manager (GTM) event tracking, Google Analytics 4 (GA4) event design, and a HubSpot CRM integration strategy for an orthopaedic consultation landing page.
